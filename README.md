# CharityLabs (unnamed) Web App

#### A web app built with Node.js on AWS

## Requirements
- Node.js
- npm
- Elastic Beanstalk CLI

## Cloning

In the desired directory run `git clone https://github.com/theAlexPatin/CharityLabs.git`

## Setup

```
$ npm install
```

## Running
```
$ npm start
```

## Deploying
```
$ eb deploy
```
